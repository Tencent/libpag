# Font Files

Place the following font files in this directory for text rendering:

- `NotoSansSC-Regular.otf` - Primary font for Chinese/CJK text
- `NotoColorEmoji.ttf` - Emoji font (optional)

These fonts can be downloaded from Google Fonts:
- https://fonts.google.com/noto/specimen/Noto+Sans+SC
- https://fonts.google.com/noto/specimen/Noto+Color+Emoji

Note: Font files are not included in the repository due to licensing and size considerations.
